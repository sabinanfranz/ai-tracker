# 50 — Test Plan

## 수동 체크리스트 (Smoke Test)

- [ ] 1. 서버가 정상 기동되는가 (`uvicorn` 실행 후 200 응답)
- [ ] 2. 메인 페이지가 브라우저에 렌더링되는가
- [ ] 3. 입력 필드에 값을 넣고 제출하면 응답이 돌아오는가
- [ ] 4. 빈 입력 제출 시 적절한 에러 메시지가 표시되는가
- [ ] 5. DB에 데이터가 정상 저장되는가
- [ ] 6. DB에서 데이터를 정상 조회할 수 있는가
- [ ] 7. LLM 호출이 타임아웃 없이 완료되는가
- [ ] 8. API 키 없이 실행 시 명확한 에러가 나오는가
- [ ] 9. 동시 요청 2건을 보내도 충돌이 없는가
- [ ] 10. `.env` 파일이 git에 포함되지 않는가

## 자동 테스트 제안

| 종류 | 대상 | 도구 |
|------|------|------|
| Unit | API 엔드포인트 | pytest + httpx |
| Unit | LangGraph 노드 | pytest |
| Integration | API → DB | pytest + SQLite in-memory |
| E2E | 전체 흐름 | playwright 또는 수동 |

## 테스트 실행 방법

```bash
# (구현 후 작성)
cd tests
pytest -v
```
